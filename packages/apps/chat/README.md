# Chat

## Overview

The Chat package (`@ait/chat`) is the landing page for AIt. It's a standalone React application built with Vite and Tailwind CSS, providing a modern, animated introduction to the platform.

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- **React 19** - UI framework
- **Vite** - Build tool and dev server
- **Tailwind CSS 4** - Utility-first styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **Vercel Analytics** - Performance monitoring

## Development

```bash
# Type checking
pnpm typecheck

# Linting
pnpm lint
pnpm lint:fix

# Preview production build
pnpm preview
```

## Deployment

The landing page is configured for deployment on Vercel. See `vercel.json` for routing configuration.
